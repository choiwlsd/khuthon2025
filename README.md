# 🌱 마이크로 비료 거래 플랫폼 "잔반걸음"

## 잔반을 거름으로!

대형 급식소·음식점·가정의 잔반을 발표해 만든 미생물 비료를 지역 농가와 거래하는 친환경 리사이클링 서비스<br>
기존의 농업 시스템을 보조: <strong>푸드웨이스트 + 순환경제 + 농촌교류</strong>

## Flow

<small><strong>"잔반걸음"</strong>은 비료를 판매하는 사람과 비료를 사는 사람 총 2가지의 유저가 존재.<br>
또한, 이용자가 직접 친환경 비료를 제작하고 해당 서비스를 이용하는 사람들에게 미생물 측정기를 제공해주었다고 가정.</small>

1️⃣ 비료 등급 및 가격 책정<br>

미생물 측정기로 친환경 비료로부터 pH, 온도, 미생물량 등의 정보를 수집하여, 해당 정보를 토대로 AI Model이 비료의 등급을 결정. 이후 비료의 등급과 무게 정보를 토대로 서비스에서 직접 가격을 책정.<br>

2️⃣ 판매글 작성 또는 조회<br>

비료를 판매하는 사용자는 사진, 제목, 설명을 입력하여 비료 판매글을 작성한다. 비료를 구매하는 사용자는 비료 판매글의 자세한 정보를 조회할 수 있다. 또, 판매 또는 구매한 비료 내역을 마이페이지에서 조회 가능하며 후기도 작성이 가능하다.

3️⃣ 잔반머니<br>

잔반을 거름으로 바꾸어 판매한 금액을 현금으로 환전할 수 있도록 한다.
